Coûts des Études

Frais de Scolarité

  • Universités publiques : 170€ - 380€/an (111,650 - 249,550 FCFA)
  • Grandes écoles : 500€ - 10,000€/an (328,500 - 6,570,000 FCFA)
  • Écoles privées : 3,000€ - 10,000€/an (1,971,000 - 6,570,000 FCFA)
  • Formation continue : 2,000€ - 15,000€/an (1,314,000 - 9,855,000 FCFA)

Les frais varient selon le niveau d'études et le type d'établissement

Logement

  • Résidence universitaire : 200€ - 400€/mois (131,400 - 262,800 FCFA)
  • Colocation : 400€ - 600€/mois (262,800 - 394,200 FCFA)
  • Studio : 500€ - 800€/mois (328,500 - 525,600 FCFA)
  • Appartement : 700€ - 1,200€/mois (459,900 - 788,400 FCFA)

Les prix varient selon la ville et le type de logement

Vie Quotidienne

  • Alimentation : 200€ - 300€/mois (131,400 - 197,100 FCFA)
  • Transport : 30€ - 75€/mois (19,710 - 49,275 FCFA)
  • Loisirs : 100€ - 200€/mois (65,700 - 131,400 FCFA)
  • Assurance : 20€ - 50€/mois (13,140 - 32,850 FCFA)

Budget mensuel moyen : 800€ - 1,200€ (525,600 - 788,400 FCFA)

Visa Étudiant

Procédure de Demande

  • Inscription dans un établissement français
  • Préparation des documents requis
  • Demande de visa au consulat
  • Validation du visa en France

Documents Requis

Passeport, photos, justificatifs financiers, attestation d'inscription

Délais

2-3 semaines pour l'obtention du visa

Frais

50€ pour le visa étudiant
