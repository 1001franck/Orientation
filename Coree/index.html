Coûts des Études

Frais de Scolarité

  • Universités publiques : 3,000,000 - 5,000,000 KRW/an (1,500,000 - 2,500,000 FCFA)
  • Universités privées : 6,000,000 - 10,000,000 KRW/an (3,000,000 - 5,000,000 FCFA)
  • Programmes en anglais : 8,000,000 - 12,000,000 KRW/an (4,000,000 - 6,000,000 FCFA)
  • Formation continue : 4,000,000 - 8,000,000 KRW/an (2,000,000 - 4,000,000 FCFA)

Les frais varient selon le niveau d'études et le type d'établissement

Logement

  • Résidence universitaire : 300,000 - 500,000 KRW/mois (150,000 - 250,000 FCFA)
  • Colocation : 400,000 - 600,000 KRW/mois (200,000 - 300,000 FCFA)
  • Studio : 500,000 - 800,000 KRW/mois (250,000 - 400,000 FCFA)
  • Appartement : 700,000 - 1,200,000 KRW/mois (350,000 - 600,000 FCFA)

Les prix varient selon la ville et le type de logement

Vie Quotidienne

  • Alimentation : 300,000 - 500,000 KRW/mois (150,000 - 250,000 FCFA)
  • Transport : 50,000 - 100,000 KRW/mois (25,000 - 50,000 FCFA)
  • Loisirs : 200,000 - 400,000 KRW/mois (100,000 - 200,000 FCFA)
  • Assurance : 30,000 - 50,000 KRW/mois (15,000 - 25,000 FCFA)

Budget mensuel moyen : 1,000,000 - 1,500,000 KRW (500,000 - 750,000 FCFA)

Visa Étudiant

Procédure de Demande

  • Inscription dans un établissement coréen
  • Préparation des documents requis
  • Demande de visa au consulat
  • Validation du visa en Corée

Documents Requis

Passeport, photos, justificatifs financiers, attestation d'inscription

Délais

2-3 semaines pour l'obtention du visa

Frais

50,000 KRW pour le visa étudiant (25,000 FCFA)
